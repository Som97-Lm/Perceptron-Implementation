This repository contains a complete implementation of the **Perceptron**, one of the earliest neural network algorithms.  
The notebook builds the perceptron from first principles: weight initialization, forward pass, training loop, error updates, and decision boundary visualization.

This is an excellent foundational exercise before learning modern neural networks (MLP, CNNs, etc.).

---

## Notebook Included
- **Neural_net_perceptron.ipynb**

This notebook demonstrates:
- What a perceptron is and how it works  
- Weight initialization  
- Activation function (step function)  
- Perceptron update rule  
- Full training loop implementation  
- Classification on toy datasets  
- Visualizing decision boundaries  
- Understanding linearly separable vs non-separable data
